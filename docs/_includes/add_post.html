Add a blog post?

  1. Create a new file in the repo with this link.

  2. Start the name of the file with the current date, for example, 2018-07-20-My_Post.md.

  3. At the beginning of the file, define the parameters like this: 

    ---
title: Post Title
author: Your Name
authorID: GitHub User
tags: [tag1, tag2]
---

    Please check the existing tags, before you create a new one.

  4. Save the post and create a pull request.
